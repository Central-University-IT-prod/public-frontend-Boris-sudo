## Запуск
- установка
```
cd frontend
npm install
npm install chart.js
```

- Запуск
```
ng serve
```

## Взаимодействие

Чтобы заливать json файлы надо перейти на вкладку `for testing` или по адресу `/testing`.
Менять время можно либо в боковом меню, либо также на вкладку `for testing`.
